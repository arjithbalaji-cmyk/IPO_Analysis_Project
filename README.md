# IPO Analysis Project (2022-2024)

## Overview
Analysis of Indian IPOs from 2022-2024, examining sector performance, oversubscription rates, and listing day returns. This project evaluates IPO strength across 16 sectors using multiple financial metrics.

## Methodology
- **Data Source:** Indian IPO market data (2022-2024)
- **Scoring System:** IPOs classified as Strong, Moderate, or Weak based on oversubscription rates and listing day performance
- **Analysis:** Pivot table analysis by sector and score category
- **Sectors Analyzed:** 16 major sectors including Agriculture, Auto, Chemicals, Energy, Finance, FMCG, Healthcare, Hospitality, Infrastructure, Logistics, Manufacturing, Real Estate, Retail, Services, Tech, and Telecom

## Key Findings
- Sector performance varies significantly across Strong/Moderate/Weak categories
- Oversubscription correlates with listing day returns
- Certain sectors show consistent strength across IPO cycles
